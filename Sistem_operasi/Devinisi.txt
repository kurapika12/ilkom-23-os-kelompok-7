Sistem operasi adalah suatu program atau software yang menjadi penghubung antara pengguna dan komputer
(hardware).Jadi dengan adanya sistem operasi itu memungkinkan penggunaan perangkat keras itu berjalan
dengan semestinya

berbagai contoh sistem operasi yang sudah sering kita kenal yakni:
1. windows
2. mac
3. Android
4. Linux
5. IOS

Fungsi Utama Sistem Operasi
Sistem operasi memiliki beberapa fungsi utama, antara lain:
1. Manajemen Proses: Mengatur eksekusi proses, termasuk penjadwalan dan pengendalian proses.
2. anajemen Memori: Mengelola penggunaan memori, termasuk alokasi dan dealokasi memori untuk aplikasi yang berjalan.
3. Manajemen Perangkat: Mengontrol dan mengatur perangkat keras, seperti printer, disk, dan jaringan.
4. Antarmuka Pengguna: Menyediakan antarmuka bagi pengguna untuk berinteraksi dengan komputer, baik melalui command line atau graphical user interface (GUI).
